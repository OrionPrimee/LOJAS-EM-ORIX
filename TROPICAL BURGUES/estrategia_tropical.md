\# 🍔 Estratégia Orion: Tropical Burgers



\## 🎯 Objetivo

Substituir o sistema Menudino por uma plataforma própria, robusta e escalável via AWS.



\## 🚀 Diferenciais Orion Prime Tech

\- \*Zero Taxas Extras:\* Sem mensalidades abusivas por pedido.

\- \*Velocidade:\* Interface otimizada para horários de pico.

\- \*Fidelização:\* Banco de dados (MySQL) para identificar os clientes VIPs de Oriximiná.



\## 🛠️ Tecnologias

\- \*Frontend:\* HTML/CSS e JS (Foco em mobile)

\- \*Backend:\* Node.js / Python

\- \*Dados:\* MySQL (Tabela de Pedidos e Clientes)

\- \*Cloud:\* AWS (Hospedagem segura)
